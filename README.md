 Expense Tracker:
 Project Overview:

This is a simple Python-based Expense Tracker developed as part of the DecodeLabs Internship Program.
The application allows users to enter multiple expense amounts, calculates the total expenditure using an accumulator, and displays the final total.

 Features:

 Add multiple expenses
 Calculates total expenses automatically
 Uses loops and user input
 Demonstrates accumulator logic
 Beginner-friendly Python project
